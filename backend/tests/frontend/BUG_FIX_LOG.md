### BUG-001: 收益曲线使用模拟数据
**问题描述**: Reports页面的收益曲线使用随机生成的模拟数据，而非真实历史收益
**严重程度**: 中
**影响范围**: 报告页面 - 收益曲线图表
**发现时间**: 2026-02-27

**代码位置**:
```typescript
// src/pages/Reports.tsx
const profitData = React.useMemo(() => {
  const data = [];
  let value = 100000;
  for (let i = 30; i >= 0; i--) {
    const date = new Date();
    date.setDate(date.getDate() - i);
    value = value * (1 + (Math.random() - 0.48) * 0.02); // 随机数据
    data.push({
      date: date.toLocaleDateString('zh-CN', { month: 'numeric', day: 'numeric' }),
      value: Math.round(value),
    });
  }
  return data;
}, []);
```

**修复建议**:
- 基于交易记录计算每日持仓市值
- 使用历史净值数据计算真实收益
- 添加时间维度切换功能

**状态**: 🟡 待修复
**预计修复时间**: 3天

**严重程度**: 低
**影响范围**: 报告页面 - 数据导入功能
**发现时间**: 2026-02-27

**代码位置**:
```typescript
// src/hooks/useDB.ts
export async function importData(jsonString: string): Promise<void> {
  const data = JSON.parse(jsonString); // 无校验
  
  if (data.funds?.length) {
    await db.funds.clear();
    await db.funds.bulkAdd(data.funds); // 直接写入
  }
  // ...
}
```

**修复建议**:
- 添加JSON Schema验证
- 导入前显示预览
- 添加错误处理和回滚机制
