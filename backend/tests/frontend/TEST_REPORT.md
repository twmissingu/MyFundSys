### 1.3 持仓自动计算

| 测试项 | 状态 | 备注 |
|--------|------|------|
| 买入更新持仓 | ✅ 通过 | 新买入创建持仓，追加买入更新成本价 |
| 卖出更新持仓 | ✅ 通过 | 卖出减少份额，份额为0时删除持仓 |
| 成本价计算 | ✅ 通过 | 使用加权平均法计算成本价 |
| 持仓市值计算 | ✅ 通过 | 份额 × 当前净值 |
| 盈亏计算 | ✅ 通过 | 市值 - 成本 |

**成本价计算公式**:
```
新成本价 = (原份额 × 原成本价 + 新买入份额 × 买入价格) / 总份额
```
