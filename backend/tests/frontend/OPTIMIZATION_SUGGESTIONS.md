## 中优先级优化

### 4. 基金详情页
**问题**: 点击基金无法查看详细信息
**建议**:
- 添加基金详情页面
- 展示基金基本信息（管理公司、成立日期等）
- 显示历史净值走势图

**页面结构**:
```
基金详情页
├── 基金基本信息
├── 当前净值和涨跌幅
├── 历史净值图表
└── 相关操作（添加交易）
```

**预估工作量**: 2天

---

### 5. 定投计划管理
**问题**: 不支持定投记录管理
**建议**:
- 添加定投计划设置页面
- 支持设置定投周期和金额
- 自动生成分批买入记录

**数据模型**:
```typescript
interface InvestmentPlan {
  id: string;
  fundId: string;
  amount: number;      // 定投金额
  frequency: 'weekly' | 'biweekly' | 'monthly';
  startDate: string;
  endDate?: string;
  executedDates: string[];
}
```

**预估工作量**: 3天

---

### 6. 交易批量导入
**问题**: 只能单条添加交易，效率低
**建议**:
- 支持CSV/Excel格式批量导入
- 提供导入模板下载
- 导入预览和错误提示

**CSV格式示例**:
```csv
基金代码,交易类型,日期,份额,价格,手续费
510300,buy,2024-01-15,1000,3.5,5
510500,sell,2024-02-20,500,5.2,3
```

**预估工作量**: 2天

---

### 7. 历史收益率计算
**问题**: 缺少年化收益率等关键指标
**建议**:
- 计算累计收益率
- 计算年化收益率（XIRR）
- 对比基准指数收益

**计算公式**:
```
年化收益率 = (期末价值/期初价值)^(365/天数) - 1
```

**预估工作量**: 2天
