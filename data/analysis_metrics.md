# 基金系统 - 分析指标定义

## 1. 概述

本文档定义基金系统的各类分析指标，包括资产配置分析、收益分析、估值分析和交易分析四大模块。

---

## 2. 资产配置分析

### 2.1 行业占比分析

#### 定义
统计投资组合中各行业基金的市值占比。

#### 计算公式
```
行业占比 = 该行业基金市值 / 投资组合总市值 × 100%
```

#### 指标说明
| 指标名 | 类型 | 说明 |
|--------|------|------|
| sector_allocation | 占比 | 按行业分类的市值占比 |
| sector_amount | 金额 | 按行业分类的持仓金额 |

#### 行业分类映射
基于 `fund_basic.sector` 字段分组统计。

---

### 2.2 市场占比分析

#### 定义
统计投资组合中各市场类型基金的市值占比。

#### 计算公式
```
市场占比 = 该市场类型基金市值 / 投资组合总市值 × 100%
```

#### 市场分类
| 分类 | 说明 | 包含基金类型 |
|------|------|-------------|
| 股票类 | A股+港股+美股 | market_type IN ('a_share', 'hk_stock', 'us_stock') |
| 债券类 | 债券基金 | market_type = 'bond' |
| 商品类 | 商品基金 | market_type = 'commodity' |
| 海外类 | QDII等 | market_type = 'overseas' |

#### 指标说明
| 指标名 | 类型 | 说明 |
|--------|------|------|
| stock_ratio | 占比 | 股票类资产占比 |
| bond_ratio | 占比 | 债券类资产占比 |
| commodity_ratio | 占比 | 商品类资产占比 |
| overseas_ratio | 占比 | 海外类资产占比 |

---

### 2.3 资产配置建议参考

基于E大的资产配置理念，推荐配置比例：

| 市场类型 | 保守型 | 稳健型 | 进取型 |
|----------|--------|--------|--------|
| A股 | 20-30% | 30-40% | 40-50% |
| 港股 | 5-10% | 10-15% | 15-20% |
| 美股 | 5-10% | 10-15% | 15-20% |
| 债券 | 40-50% | 25-35% | 10-20% |
| 商品 | 5-10% | 5-10% | 5-10% |
| 海外其他 | 5-10% | 5-10% | 5-10% |

## 3. 收益分析

### 3.1 总收益

#### 定义
投资组合的累计收益金额。

#### 计算公式
```
总收益 = 当前市值 - 总投入成本
       = Σ(持仓基金市值) - Σ(买入金额) + Σ(卖出金额)
```

#### SQL参考
```sql
SELECT 
    SUM(market_value - cost_basis) as total_return
FROM position_snapshot
WHERE snapshot_date = '2026-02-27';
```

---

### 3.2 收益率

#### 定义
投资回报率，以百分比表示。

#### 计算公式
```
收益率 = (当前市值 - 总投入成本) / 总投入成本 × 100%
```

---

### 3.3 年化收益率 (IRR)

#### 定义
考虑资金时间价值的投资收益率。

#### 计算公式
使用XIRR（扩展内部收益率）算法：
```
NPV = Σ(CF_t / (1 + r)^((t - t0) / 365)) = 0
求解 r 即为年化收益率
```

其中：
- CF_t: 第t天的现金流（买入为负，卖出为正，当前市值为正）
- t: 日期
- t0: 起始日期
- r: 年化收益率

---

### 3.4 最大回撤 (Max Drawdown)

#### 定义
从峰值到谷底的最大跌幅，衡量投资风险。

#### 计算公式
```
最大回撤 = max((峰值 - 谷值) / 峰值) × 100%
```

#### 计算步骤
1. 计算每日累计收益率曲线
2. 找到每个峰值后的最大跌幅
3. 取所有跌幅中的最大值

#### SQL参考
```sql
-- 计算历史最大回撤
WITH daily_value AS (
    SELECT snapshot_date, SUM(market_value) as total_value
    FROM position_snapshot
    GROUP BY snapshot_date
),
peak_value AS (
    SELECT 
        snapshot_date,
        total_value,
        MAX(total_value) OVER (ORDER BY snapshot_date) as peak
    FROM daily_value
)
SELECT 
    MAX((peak - total_value) / peak) as max_drawdown
FROM peak_value;
```

---

### 3.5 夏普比率 (Sharpe Ratio)

#### 定义
风险调整后的收益率，衡量每单位风险获得的超额收益。

#### 计算公式
```
夏普比率 = (投资组合收益率 - 无风险收益率) / 投资组合波动率
```

#### 参数说明
- 无风险收益率：通常取10年期国债收益率，约2-3%
- 投资组合波动率：日收益率的标准差 × √252

---

### 3.6 收益指标汇总

| 指标名 | 英文名 | 说明 | 计算频率 |
|--------|--------|------|----------|
| 总收益 | Total Return | 累计盈亏金额 | 每日 |
| 收益率 | Return Rate | 累计收益率 | 每日 |
| 年化收益 | Annualized Return | 年化收益率 | 每日 |
| 最大回撤 | Max Drawdown | 历史最大跌幅 | 每日 |
| 夏普比率 | Sharpe Ratio | 风险调整后收益 | 每月 |
| 卡玛比率 | Calmar Ratio | 收益与最大回撤比 | 每月 |
| 胜率 | Win Rate | 盈利交易占比 | 每月 |

## 4. 估值分析

### 4.1 当前估值位置

#### 定义
基于PE/PB百分位判断当前估值水平。

#### 估值等级划分
| 等级 | PE百分位 | PB百分位 | 说明 |
|------|----------|----------|------|
| 极度低估 | < 10% | < 10% | 历史低位，适合重仓 |
| 低估 | 10% - 30% | 10% - 30% | 偏低位置，适合建仓 |
| 合理偏低 | 30% - 50% | 30% - 50% | 合理区间，正常持有 |
| 合理偏高 | 50% - 70% | 50% - 70% | 合理区间，谨慎持有 |
| 高估 | 70% - 90% | 70% - 90% | 偏高位置，考虑减仓 |
| 极度高估 | > 90% | > 90% | 历史高位，建议清仓 |

---

### 4.2 历史估值对比

#### 指标定义
| 指标名 | 说明 |
|--------|------|
| pe_current | 当前PE |
| pe_min_5y | 近5年PE最小值 |
| pe_max_5y | 近5年PE最大值 |
| pe_avg_5y | 近5年PE平均值 |
| pe_median_5y | 近5年PE中位数 |
| pb_current | 当前PB |
| pb_percentile | 当前PB百分位 |

---

### 4.3 估值温度

#### 定义
综合PE和PB百分位的估值热度指标。

#### 计算公式
```
估值温度 = (PE百分位 + PB百分位) / 2 × 100
```

#### 温度区间
| 温度 | 区间 | 操作建议 |
|------|------|----------|
| 冰冷 | 0-10 | 极度低估，大胆买入 |
| 寒冷 | 10-30 | 低估，积极买入 |
| 凉爽 | 30-50 | 合理偏低，正常买入 |
| 温暖 | 50-70 | 合理偏高，谨慎买入 |
| 炎热 | 70-90 | 高估，停止买入/考虑卖出 |
| 滚烫 | 90-100 | 极度高估，果断卖出 |

## 5. 交易分析

### 5.1 买卖频率分析

#### 指标定义
| 指标名 | 说明 | 计算公式 |
|--------|------|----------|
| trade_count | 总交易次数 | COUNT(*) |
| buy_count | 买入次数 | COUNT(*) WHERE trade_type=1 |
| sell_count | 卖出次数 | COUNT(*) WHERE trade_type=2 |
| trade_freq | 交易频率 | 交易次数 / 投资天数 |
| avg_trade_interval | 平均交易间隔 | 投资天数 / 交易次数 |

---

### 5.2 成本分析

#### 指标定义
| 指标名 | 说明 | 计算公式 |
|--------|------|----------|
| total_fee | 总手续费 | SUM(fee) |
| avg_fee_rate | 平均费率 | SUM(fee) / SUM(amount) |
| fee_by_platform | 分平台手续费 | SUM(fee) GROUP BY platform |
| cost_impact | 成本影响 | 总手续费 / 总投入成本 |

---

### 5.3 交易效率分析

#### 指标定义
| 指标名 | 说明 | 计算公式 |
|--------|------|----------|
| win_trade_count | 盈利交易次数 | 卖出时收益>0的次数 |
| loss_trade_count | 亏损交易次数 | 卖出时收益<0的次数 |
| win_rate | 胜率 | 盈利次数 / 总卖出次数 |
| avg_profit | 平均盈利 | 盈利总额 / 盈利次数 |
| avg_loss | 平均亏损 | 亏损总额 / 亏损次数 |

## 6. 综合分析指标

### 6.1 投资组合评分

基于多维度计算投资组合健康度评分：

| 维度 | 权重 | 评分标准 |
|------|------|----------|
| 资产配置合理性 | 25% | 是否符合目标配置比例 |
| 估值水平 | 25% | 整体估值百分位 |
| 收益表现 | 20% | 年化收益、最大回撤 |
| 风险控制 | 20% | 集中度、行业分散度 |
| 成本控制 | 10% | 手续费占比 |

#### 评分等级
| 分数 | 等级 | 说明 |
|------|------|------|
| 90-100 | 优秀 | 配置合理，风险可控 |
| 80-89 | 良好 | 整体健康，略有优化空间 |
| 70-79 | 一般 | 需要关注，适当调整 |
| 60-69 | 较差 | 存在问题，建议调整 |
| < 60 | 危险 | 风险较高，急需调整 |
