# E大基金代码提取项目

## 项目概述

本项目从E大（ETF拯救世界）的历史文章中提取所有提到的基金代码，建立完整的基金池，用于后续基金分析和投资决策支持。

## 数据来源

- **文章目录**: `/root/.openclaw/workspace/agents/fund-agent/data/etf-savior/chinaetfs-full`
- **文章数量**: 365篇
- **时间跨度**: 2010-2025年

## 输出文件

### 主要输出文件（推荐使用）

| 文件名 | 说明 | 格式 |
|--------|------|------|
| `fund_list_complete.csv` | 完整基金清单（含名称、分类、市场等） | CSV |
| `fund_classification_complete.csv` | 基金分类表 | CSV |
| `fund_classification_complete.json` | 基金分类表（JSON格式） | JSON |
| `fund_codes_complete.txt` | 基金代码文本列表 | TXT |

### 原始提取文件

| 文件名 | 说明 |
|--------|------|
| `fund_list.csv` | 仅从文章提取的基金清单（38只） |
| `fund_classification.csv` | 仅从文章提取的分类表 |
| `fund_classification.json` | 仅从文章提取的分类表（JSON） |

## 提取脚本

### 使用说明

```bash
# 运行完整版提取脚本
cd /root/.openclaw/workspace/projects/fund-system/data/fund-list
python3 extract_funds_complete.py
```

### 脚本说明

- `extract_funds.py` - 基础版提取脚本
- `extract_funds_v2.py` - 优化版
- `extract_funds_v3.py` - 增加上下文分析
- `extract_funds_v4.py` - 改进分类逻辑
- `extract_funds_complete.py` - 完整版（推荐）
